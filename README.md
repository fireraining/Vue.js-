# vue-
项目是基于Vue.js，成品是一个移动端的音乐播放器，来源于imooc的实战课程。自己动手实践并加以修改拓展。 项目的大致流程是Vue-cli构建开发环境，分析需求，设计构思，规划目录结构，开始编码。
技术栈：
Vue

Vuex

Vue-Router

Vue-cli

Stylus

Axios

ESlint

Better-scroll

开发环境
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# run e2e tests
npm run e2e

# run all tests
npm test

生产环境
# build for production with minification
npm run build
# run
node prod.server.js
